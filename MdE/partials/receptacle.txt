MdE Réceptacle




* PaveNum : Pavé numérique type standard. Chiffre de 0-9, bouton valider, bouton annuler.
   * rentrerCode(n:number) : Le code n est vérifié.
   * modifierCode(n:number) : Le code n est fixé en tant que nouveau code.




* capteurBadge :  Capteur de badge pour détecter la présence du badge de l’utilisateur (ou celui du technicien)
   * vérificationBadge(b:Idbadge) : La valeur du badge est vérifiée.




* diode : Diode pouvant prendre les couleurs verte et rouge.
   * diodeVerte(): Demande à la diode de devenir verte pour 3 secondes
   * diodeRouge(): demande à la diode de devenir rouge pour 3 secondes




* panneauAffichage : Pour l’affichage d’informations sur l’état du système
   * msgAcceuil(): Demande d’affichage du message d’accueil.
   * msgCodeIncorrect(e:boolean): Si e=vrai, demande d’affichage d’un message indiquant que le code rentré est incorrect. Si e=faux, demande du réceptacle d’effacer le message.
   * msgBadgeIncorrect(e:boolean): similaire à msgCodeIncorrect().
   * msgModifierCode(e:boolean):  similaire à msgCodeIncorrect(). L’effacement est suivit d’une demande d’affichage d’un massage demandant la fermeture.
   * msgHorsService(e:boolean): similaire à msgCodeIncorrect().




* supportDrone : 
   * dronePresent(): Le support indique que le drône est présent.
   * bloquerDrone(e:boolean): Si e=vrai, Demande au support de bloquer le drône 
                Si e=faux, demande au support de relâcher le drône




* porteLaterale : Porte latérale du réceptacle. Est utilisée par le client pour récupérer les colis. Ne peut être ouverte que manuellement.
   * porteFerme(): signale au réceptacle que la prote vien d’être fermée
   * bloquerPorte(): demande du réceptacle de bloquer la porte latérale.
   * débloquerPorte(): demande du réceptacle de débloquer la porte latérale.




* panneauSuperieur : Panneau supérieur du réceptacle. Utilisé par le drone pour déposer les colis. Peut être ouverte électroniquement ou manuellement.
   * panneauFerme(): signale au réceptacle que le panneau supérieur vien d’être fermé
   * bloquerPanneau(): demande du réceptacle de bloquer le panneau supérieur.
   * debloquerPanneau(): demande du réceptacle de débloquer le panneau supérieur.
   * ouvrirPanneau(): demande du réceptacle d’ouverture du panneau.
   * fermerPanneau(): demande du réceptacle de fermer le panneau supérieur.




* capteurRFID : Un capteur RFID permettant de vérifier la présence à l’intérieur du réceptacle
   * colisPresent(): le capteur indique qu’il a détecté la présence d’un colis à l’intérieur du réceptacle


* systemeDrone : Correspond au système du drone en liaison avec le réceptacle.
   * verificationColis(idDestinataire:IdRFID) : Signal du drone demandant la vérification de la correspondance entre un colis et le réceptacle
   * ouverturePanneau() : Signal du drone demandant l’ouverture du panneau supérieur du réceptacle.
   * fermeturePanneau() : Signal du drone demandant la fermeture du panneau supérieur du réceptacle.
   * timeoutVerification() : Signal du drone indiquant que celui-ci n’a pas reçu la confirmation de l’action demandée dans le temps voulu.
   * receptaclePlein() : Signal du réceptacle indiquant au drone que le réceptacle est plein.
   * colisCorrect(e:bool) : Si e=vrai, signale que le colis correspond bien à ce réceptacle. Si e=faux, signale que le colis ne correspond pas à ce réceptacle.
   * signalReceptacle() : Signal du réceptacle pour indiquer  au drone la position du réceptacle pour que celui-ci se pose.
   * sendConfirmation(msg:ConfirmMsg) : signal du receptacle pour indiquer qu’une action a bien été réalisée. Cette action est indiqué par le msg (sous la forme d’une énumération






* systemeClient :
   * SignalerReception : Signal du réceptacle au système client pour indiquer qu’il y a un colis à récupérer dans le réceptacle.
